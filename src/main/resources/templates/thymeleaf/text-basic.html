데이터 출력

  • text1 = Hello Thymeleaf!!
  • th:text 를 이용한 text1 출력 결과 =
  • 컨텐츠 안에서 직접 출력하기: [[${text1}]]으로 text1 출력 결과 = [[${text1}]]

text & utext

  • text2 =
  • th:text 로 그냥 출력했을때 결과 =
  • th:utext 를 이용해서 출력했을때 결과=

[[...]] & [(...)]

  • text2 =
  • [[...]] 를 사용한 출력 결과 = [[${text2}]]
  • [(...)] 를 사용한 출력 결과 = [(${text2})]
  • th:inline="none" = 는 이 테그안에서 타임리프가 해석하지 말라는 것이다.
